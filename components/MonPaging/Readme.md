## 简单分页组件

### 参数

| 参数名 | 类型 | 是否必须 | 描述 |
| -------- | -------- | -------- | -------- |
| value     | Number     | 否     | 当前分页     |
| pageSize  | Number     | 否     | 每页显示记录数，默认 10     |
| count     | Number     | 否     | 总记录数     |


### 事件

| 参数名  | 描述 | 回调参数 |
| -------- | -------- | -------- |
| change   | 页码改变的回调，参数是改变后的页码及每页条数  | Function(page)  |
