## 图片

### 模板语法

```vue
<mon-img :src="`http://kab.test/public/static/images/logo.png`" :width="`400px`"/>
```

### API

| 参数名 | 类型 | 是否必须 | 描述 |
| -------- | -------- | -------- | -------- |
| src     | String    | 是     | 图片路径	     |
| alt     | String    | 否     | 图片加载失败提示文案     |
| width   | String    | 否     | 图片宽度，默认100%     |
| height  | String    | 否     | 图片宽度，默认100%     |
